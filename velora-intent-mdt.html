The case

Ravi Shankar Iyengar, 58, CKD on PD

Six encounters across three departments in 80 days. Each chart was clinically complete on its own. The pattern that mattered — KT/V dropping, BNP rising, AF emerging — was the kind of pattern only visible when all three records sit in front of the same person at the same time. The hospital's existing tools did not make that possible inside a 15-minute OPD slot.

Nephrology
On file in Nephrology's chart:
  • KT/V trend: 1.38 → 1.62 → 1.21
  • PD prescription intensified to 4 exchanges/day
  • Anaemia of CKD managed (Hb 8.4 g/dL)
Cardiology
On file in Cardiology's chart:
  • Holter: 3 nocturnal AF episodes
  • Apixaban initiated (CKD-adjusted dose)
  • Echo: EF 38%, LV hypertrophy
General Medicine
On file in General Medicine's chart:
  • ABPM: non-dipper pattern
  • BP 174/100 on 4 antihypertensives
  • Grade III hypertensive retinopathy

Each chart was clinically complete. The cross-specialty correlation only emerged when all three records were viewed together — which the hospital's existing tools did not enable in time.

"Pattern is clear and recurrent. Dialysis inadequacy is the root cause of both admissions. This correlation is not visible within any single departmental record and requires cross-encounter data synthesis to establish." — Pathologist's impression, March 2026

What guidelines say at this state.

Velora cites what guideline bodies recommend reviewing at this trajectory. It does not predict, prescribe, or advise.

+ISN-India 2020 · PD adequacy
Patients with KT/V < 1.7 sustained should be reviewed for prescription intensification, residual function reassessment, or modality discussion.
ISN-India PD §4.2 · §5.1 (review triggers)
+KDIGO 2012 · Anaemia in CKD
Hb < 10 g/dL on PD warrants ESA review and iron-status workup before further escalation.
KDIGO Anaemia 2012 §3.2 · §3.4
+ESC/ESH 2023 · Resistant HTN
≥ 4 antihypertensives with persistent ABPM elevation: secondary causes review (renal artery stenosis, primary aldosteronism, OSA).
ESC/ESH 2023 §9.1 · §9.4 (resistant HTN workup)
+ESC 2024 · AF anticoagulation
CHA₂DS₂-VASc ≥ 2 with CKD eGFR < 15 — apixaban remains preferred DOAC; bleed risk should be re-scored at each MDT.
ESC 2024 AF §6.3 · §10.2

Indian + global guidelines

Same patient. Different hospital. Different rule wins.

For PD adequacy, two guideline bodies cover the same clinical area: KDIGO (international, US-led) and ISN-India (Indian Society of Nephrology, 2020). Targets differ slightly because Indian dialysis-population baselines differ.

The hospital's deployment config picks which one wins for that hospital's clinicians. Same code, same rule library, different hospital config.
